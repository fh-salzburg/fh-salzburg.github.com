Why are we here?

Git is used extensively in both the bachelor and masters degree program multimedia­technology. We set up this account on github for the whole university, and the account multimediatechnology to accommodate both student and research projects.

I'm a student, how can I use this?

Get a normal github account. Then write to Brigitte Jellinek (from your fh e-mail adress), stating the name of the repository you need. She'll add you as a member and create the repository for you.

Can I see the projects?

You can browse the source code of the projects right here on github. Or you can look at the finished projects in our portfolio site.
